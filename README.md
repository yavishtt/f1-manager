# 🏎️ The Pit Wall — F1 Manager Website

A Formula 1 information and management website built as a Web Programming Lab project.
Inspired by CricBuzz but for Formula 1 — live standings, driver profiles, circuit maps, and more.

## 📄 Pages

| Page | Description |
|------|-------------|
| `index.html` | Landing page with hero section, live ticker, driver previews, race schedule |
| `drivers.html` | All 20 F1 drivers with photos, stats, search & filter, click-to-open modal |
| `teams.html` | Constructors standings table with animated points bars and team profiles |
| `races.html` | Full 2025 race calendar with countdown timer and filters |
| `circuits.html` | 14 circuits drawn in game style with interactive animated racing line modal |
| `login.html` | Login page with form validation |
| `register.html` | Registration with password strength meter |
| `contact.html` | Contact and feedback form with star rating |

---

## ⚙️ Tech Stack

- **HTML5** — structure and semantics
- **CSS3** — custom design system, animations, responsive layout
- **Vanilla JavaScript** — interactivity, Canvas API for circuit maps, DOM manipulation
- **Google Fonts** — Barlow Condensed, Barlow, Share Tech Mono
- **HTML Canvas API** — all circuit track maps drawn in code (no images)

> No frameworks. No libraries. No backend. Pure HTML/CSS/JS only.

---

## ✨ Features

- 🗺️ 14 F1 circuits drawn as game-style track maps using Canvas API
- 🎬 Animated racing line with sector colours (yellow/green/purple) on click
- 🧑‍🤝‍🧑 Driver cards with real photos, search, team filter, and modal profiles
- 🏢 Team constructors standings with animated progress bars and SVG logos
- ⏱️ Live countdown timer to the next race
- 📱 Fully responsive design
- 🖱️ Custom animated cursor
- 🌙 Dark theme with noise texture overlay
